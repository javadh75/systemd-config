# Security Policy

## Supported versions

Only the latest release receives security fixes. The library has no
runtime dependencies; its attack surface is parsing untrusted unit-file
input (fuzzed continuously in CI).

| Version | Supported |
| ------- | --------- |
| latest (v0.4.x) | ✅ |
| older | ❌ |

## Reporting a vulnerability

Please report vulnerabilities privately via GitHub's security advisories:

- <https://github.com/javadh75/systemd-config/security/advisories/new>

Do not open a public issue for security reports. You should receive a
response within a week. Once a fix is released, the advisory is published
and credited to the reporter unless they prefer otherwise.
